# 整型与字符串的转换

## 整型转字符串

```cpp
int a = 123;
//不只是整型，其他类型也可以
string b = to_string(a);
```

## 字符串转整型、浮点型

```cpp
int main() {
    string a = "123";
    string b = "1.5";
    int num1 = atoi(a.c_str());
    float num2 = atof(b.c_str());
    cout << "转成整数  :" << num1 << endl;
    cout << "转成浮点数:" << num2 << endl;
    return 0;
}
```
